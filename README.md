# @keeb/discord

[Swamp](https://github.com/systeminit/swamp) extension for Discord webhook messaging.

## Models

### `discord/webhook`

Post messages to a Discord channel via webhook URL.

| Method | Description |
|--------|-------------|
| `send` | Send a message (requires `content`, optional `username`) |

**Global arguments:** `webhookUrl`

## Install

```bash
swamp extension pull @keeb/discord
```
